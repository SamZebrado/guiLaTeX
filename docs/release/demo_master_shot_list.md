## 推荐展示结构

```
第一幕：Web 独立应用（~2.5 分钟）
  └─ 纯浏览器体验，零依赖，展示核心编辑能力
```

## 第三幕：跨端 LaTeX 闭环

### Shot 14：Qt 导出 → Web 导入（30s）

- **演示目标**：展示跨端一致性
- **录制步骤**：
  1. 在 Qt 端导出 conforming tex
  2. 通过 Python bridge 转换为 Web 可导入的 JSON
  3. 在 Web 端导入 → 展示文档恢复
- **对外可说**："Qt 导出的 LaTeX 可通过 bridge 导入 Web 端，核心几何和内容字段保留"
- **不该说满**：不要说"跨端完全无损"（字体和层级字段有转换）
- **是否必须在真实环境中录**：是（需要 Python bridge 环境）

### Shot 15：差异分析展示（15s）

- **演示目标**：展示跨端差异可追溯
- **录制步骤**：展示 `web_import_from_qt_diff.json` 差异报告
- **对外可说**："跨端差异可字段级追溯"
- **不该说满**：不要说"差异为零"
