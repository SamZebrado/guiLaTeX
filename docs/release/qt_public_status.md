# Qt 端公开状态

**更新日期**：2026-04-19（最新裁决更新）
**定位**：**原型阶段**（不作为当前主完成判据）

---

## 总体评价

Qt 端核心闭环已跑通（导出 conforming LaTeX → 重新导入 → 字段级差异对比），但**多项功能尚未收口**，不作为当前主完成判据。Web 端是当前主展示线。

**这不是最终完整版。**

## 已验证的功能

| 功能 | 状态 | 说明 |
|------|------|------|
| 元素选择与属性编辑 | ✅ | 点击元素，属性面板实时更新 |
| 旋转 | ⚠️ 受限 | **仅能通过配置面板调节**，无直接旋转控件 |
| 复制/粘贴 | ✅ | 基本操作正常 |
| 导出 conforming LaTeX | ✅ | 通过 Core 导出器生成 .tex |
| 导入 LaTeX | ✅ | 导入 .tex 后恢复文档结构 |
| Roundtrip 主闭环 | ✅ | Qt → Core → tex → Core → Qt，字段级差异可追溯 |
| Core 集成 | ✅ | Export IR 中间层已接入 |

## 未收口问题（必须公开说明）

| 问题 | 严重程度 | 说明 |
|------|---------|------|
| ⚠️ 旋转仅配置面板 | 高 | **无直接旋转控件**，只能通过属性面板手动输入角度 |
| ⚠️ 图层功能未体现 | 高 | 图层功能在编辑界面中**看不出效果**，无法作为展示功能 |
| ⚠️ 导出 PDF 闪退 | 高 | **导出 PDF 会造成应用闪退**，此功能不可用 |
| ⚠️ font_family Core gap | 中 | 中文字体硬编码为 SimSun，英文字体硬编码为 Times New Roman，忽略 Qt 模型值 |
| ❌ 保存/打开项目 | — | 功能未实现 |
| ❌ 变换菜单 | — | 功能未实现 |
