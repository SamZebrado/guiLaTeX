# Qt 演示录制脚本

日期：2026-04-18

## 录制目标
展示 Qt 界面的核心功能，重点突出 roundtrip 能力和基本编辑体验。

## 拍摄清单

### 1. 开场介绍（30秒）
**录制目标**：介绍 Qt 界面的整体布局
**操作步骤**：
1. 启动 Qt 应用
2. 展示顶部工具栏分组
3. 展示左侧画布和右侧属性面板
4. 简要说明各区域功能

**建议时长**：30秒
**对外可以怎么说**：
- "这是 guiLaTeX 的 Qt 桌面版本，采用了统一的 UI 模式 v1"
- "左侧是画布区域，右侧是属性编辑面板"
- "顶部工具栏按功能分组，操作清晰直观"

**不该说满的话**：
- 不要说 UI 是最终版
- 不要说所有功能都已实现

### 2. 元素选择与属性编辑（60秒）
**录制目标**：展示元素选择和属性编辑能力
**操作步骤**：
1. 点击画布上的标题元素
2. 在右侧属性面板修改文本为"演示标题"
3. 修改字体大小为 28
4. 观察画布上的实时更新
5. 选择段落元素，修改文本内容

**建议时长**：60秒
**对外可以怎么说**：
- "选择元素后，右侧属性面板会显示详细属性"
- "可以实时编辑文本内容和字体大小"
- "修改会立即反映在画布上"

**不该说满的话**：
- 不要说所有属性都可编辑
- 不要说属性面板功能齐全

### 3. 旋转功能（45秒）
**录制目标**：展示元素旋转功能
**操作步骤**：
1. 选择段落元素
2. 在属性面板将旋转角度改为 45 度
3. 观察元素旋转效果
4. 改回 0 度，再改为 -30 度
5. 展示不同角度的旋转效果

**建议时长**：45秒
**对外可以怎么说**：
- "支持对元素进行任意角度的旋转"
- "旋转效果会实时显示在画布上"
- "旋转信息会保存到模型中"

**不该说满的话**：
- 不要说旋转精度很高
- 不要说旋转支持所有元素类型

### 4. 复制/粘贴功能（45秒）
**录制目标**：展示元素复制粘贴能力
**操作步骤**：
1. 选择标题元素
2. 点击编辑菜单 -> 复制
3. 点击编辑菜单 -> 粘贴
4. 观察新元素出现在下方
5. 选择新元素，在属性面板查看其 ID

**建议时长**：45秒
**对外可以怎么说**：
- "支持元素的复制和粘贴"
- "粘贴时会自动生成新的 ID"
- "新元素会自动偏移位置，避免重叠"

**不该说满的话**：
- 不要说复制粘贴支持复杂对象
- 不要说复制粘贴速度很快

### 5. 导出 LaTeX 功能（60秒）
**录制目标**：展示导出为 conforming LaTeX 的能力
**操作步骤**：
1. 点击文件菜单 -> 导出 LaTeX
2. 选择保存位置，文件名设为"demo_export"
3. 打开生成的 .tex 文件
4. 展示文件包含 IR 元数据
5. 展示生成的 LaTeX 代码结构

**建议时长**：60秒
**对外可以怎么说**：
- "支持导出为 conforming LaTeX 文件"
- "导出的文件包含完整的 IR 元数据"
- "可以直接用于 Web 端导入"

**不该说满的话**：
- 不要说导出支持所有复杂元素
- 不要说字体选择完全保住

### 6. 导入 LaTeX 功能（60秒）
**录制目标**：展示从 conforming LaTeX 导入的能力
**操作步骤**：
1. 点击文件菜单 -> 导入 LaTeX
2. 选择刚才导出的"demo_export.tex"
3. 观察元素被正确导入
4. 选择导入的元素，验证其属性
5. 对比导入前后的效果

**建议时长**：60秒
**对外可以怎么说**：
- "支持导入 conforming LaTeX 文件"
- "能正确恢复元素的位置、大小和旋转信息"
- "导入过程简单直观"

**不该说满的话**：
- 不要说导入支持任意 LaTeX 文件
- 不要说所有字段都完全保住

### 7. 总结（30秒）
**录制目标**：总结 Qt 当前状态
**操作步骤**：
1. 回到主界面
2. 简要回顾演示的功能
3. 说明当前状态和未来计划

**建议时长**：30秒
**对外可以怎么说**：

- "支持与 Web 端的 roundtrip 流程"
- "已进入 v1-candidate 阶段，准备进行进一步的完善"

**不该说满的话**：

- 不要说所有功能都已实现
- 不要说没有任何问题
