测试导出 IR 功能

1. 点击瞬移问题测试

测试步骤：

  1. 打开 Web 原型
  2. 点击任意文本元素
  3. 观察元素是否会发生位置跳动

预期结果：点击元素后，元素位置保持不变，只显示选中状态

2. 多选旋转问题测试

测试步骤：

  1. 打开 Web 原型
  2. 勾选"多选模式"
  3. 点击多个元素进行选择
  4. 使用旋转手柄或旋转滑块旋转选中的元素
  5. 观察是否所有选中的元素都同时旋转

预期结果：所有选中的元素都同时旋转相同的角度

3. 导出 IR 功能测试

测试步骤：

  1. 打开 Web 原型
  2. 点击"导出 IR"按钮
  3. 检查下载的 guilatex_ir.json 文件

预期结果：下载的 JSON 文件包含所有元素的 IR 格式数据，包含所有必需字段

4. 字体安全列表测试

测试步骤：

  1. 打开 Web 原型
  2. 打开浏览器开发者工具
  3. 查看控制台输出的字体信息

预期结果：控制台输出包含字体栈信息，只包含开源/免费可商用字体

5. 文件功能文案测试

测试步骤：

  1. 打开 Web 原型
  2. 检查顶部的文件操作按钮

预期结果：按钮文案清晰诚实，包含"打开项目"、"保存项目"、"导出截图"、"导出 IR"
