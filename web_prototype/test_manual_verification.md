# 手动验证测试

## 测试目标
验证以下两个问题是否已修复：
1. 点击对象瞬移问题
2. 多选旋转是否真的作用于多个对象

## 测试环境
- 本地服务器地址: http://localhost:8007/web_prototype/index.html
- 浏览器: 任意现代浏览器

## 测试步骤

### 测试 1: 点击瞬移问题

#### 场景 1: 未旋转对象，点击中心点，不拖动
1. 打开测试页面
2. 找到页面上的标题元素（"标题：可视化编辑 LaTeX 文档 (文本框)"）
3. 点击元素的中心点
4. 观察元素位置是否变化
5. 记录结果

#### 场景 2: 未旋转对象，点击右下区域，不拖动
1. 打开测试页面
2. 找到页面上的标题元素（"标题：可视化编辑 LaTeX 文档 (文本框)"）
3. 点击元素的右下区域
4. 观察元素位置是否变化
5. 记录结果

#### 场景 3: 点击后立即开始拖动
1. 打开测试页面
2. 找到页面上的标题元素（"标题：可视化编辑 LaTeX 文档 (文本框)"）
3. 点击元素的任意位置
4. 立即开始拖动元素
5. 观察元素是否在拖动开始瞬间跳变
6. 记录结果

### 测试 2: 多选旋转功能

1. 打开测试页面
2. 勾选"多选模式"复选框
3. 点击标题元素（"标题：可视化编辑 LaTeX 文档 (文本框)"）
4. 点击作者元素（"作者：某某 / 博士生原型测试 (文本框)"）
5. 确认两个元素都被选中（显示红色边框）
6. 使用旋转滑块将角度调整为 45 度
7. 观察两个元素是否都旋转了 45 度
8. 记录结果
