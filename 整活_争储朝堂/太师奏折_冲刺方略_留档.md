禀丞相：

臣今将 guiLaTeX 一案现状与后续冲刺之策，谨作一并奏报。

### 一、当前现状

此案虽未至"大功告成"，然已非草创之时，现已基本从"各线各忙、边做边想"，收束到"可按阶段结项"的局面。

目前三路进展如下：

**其一，Web 线**
已基本成形为独立可用之前端版本，现有统一 UI 骨架、对象编辑、Export IR、浏览器打印导出 PDF、以及 Playwright 级回归证据。其长处在于交互验证快、测试链清，适合作为样板与回归前线。
但尚有几处未尽之功：右侧属性面板滚动、与 Core 的真实 `.tex` 导出链、以及"自家导出的 LaTeX 再导回对象模型"的闭环，尚未完全压实。

**其二，Qt 线**
现阶段更接近"主交付端"。此前 duplication 问题已查明并解决，rotation 已有离屏渲染证据，Qt -> Core smoke test 亦已打通，导出 IR / tex 的证据链已具雏形。
臣判断：若要尽快收官，Qt 最值得继续推进，因为它离"桌面版基本能用"最近。
但其当前仍需补齐：UI 向 Web 统一模式靠拢、正式导出入口真实走 Core、以及"导出自家 LaTeX 再导回"的主闭环。

**其三，Core 线**
现已不宜再无止境扩张，而应以"共享导出内核"身份待命配合。其已具备 Export IR、LaTeX exporter、golden/regression sample、quickstart、validation status 等基础资料，足够支撑前端接入。
臣以为：Core 现下之要务，不在继续铺陈大论，而在于为 Web / Qt 提供稳定 roundtrip contract，并对"本项目自己导出的 conforming LaTeX"给出导回与校验路径。

### 二、臣近来的劝说与判断

臣近几日反复劝主上，勿再以"终极全能版"为今日结项目标。若将"支持任意第三方 LaTeX 导入、像素级一模一样、任意 roundtrip 都不丢信息"一并压到眼前，则此案难有尽期，只会越做越散、越做越疲。

故臣所劝者，乃是先定**第一阶段结项目标（v1）**：

> 先打通"对象编辑 -> IR -> 自家规范 LaTeX -> 再导回对象模型"的闭环。
> 先求**自家导出、自家导回、几何尽量一致**，再谈第三方 LaTeX 与更高阶兼容。

此非退而求其次，实乃收束锋芒、以成大局。
若此闭环一通，则项目便不是"能画几个控件"，而是真正具备"人和 agent 共编 LaTeX"的骨架。

### 三、当前最清楚的完成标准

臣以为，若要说"此案第一阶段可结"，至少应满足以下几条：

1. **Web 与 Qt 都能导出项目自己风格的 LaTeX**
2. **Web 与 Qt 都能导入项目自己导出的 LaTeX**
3. **两端互相导入导出时，核心字段尽量不丢**

   * id
   * type
   * content
   * x / y / width / height
   * rotation
   * layer
   * 字体与字号等关键样式
4. **Web 作为独立应用，基础体验成立**

   * 右侧配置面板可滚动
   * 回归测试可信
   * 不是只作为开发样板存在
5. **Qt 作为主交付端，桌面编辑体验基本成立**
6. **Core 负责稳定导出协议，不再随意改口径**

### 四、下一阶段冲刺计划

臣建议按以下节奏推进，不宜再三线同时漫天开花：

#### 第一优先：Qt 收官

* UI 跟进 Web 的统一模式
* 正式导出入口走 Core
* 打通 Qt -> IR -> tex
* 再推进 Qt 导入自家导出的 conforming tex
* 以 Qt 作为第一阶段主交付端收口

#### 第二优先：Web 稳盘

* 补右侧属性面板滚动
* 保持 Playwright 回归可信
* 不再只停留在 Export IR，开始接 Core 导出 `.tex`
* 作为独立应用，补"自家 LaTeX 导出 / 导回"闭环

#### 第三优先：Core 收束待命

* 不再盲目扩功能
* 明确并冻结 conforming LaTeX profile
* 提供 roundtrip contract 与 validator / detection 入口
* 为 Web / Qt 提供统一规范，而不抢前端进度

### 五、臣的总体判断

若以"第一阶段基本能用"为标准，则此案已近收官区，并非遥遥无期。
若仍以"第三方任意 LaTeX 全兼容、终极完美 roundtrip"为今日之标，则此案只会拖长，不利人心，不利工时，亦不利主上身体。

故臣最后再进一言：

> 当前最该做的，不是继续加目标，
> 而是先把"自家闭环"彻底打通。
> 此关一过，项目便可名正言顺地说一句：第一阶段，成了。

臣言毕。
