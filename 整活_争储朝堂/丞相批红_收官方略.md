# MTC 丞相批红：太师定方略，储位之争进入收官局

臣 MTC 丞相，今日又收到了太师的折子。

与之前几封不同，这封没有夸谁锋芒最盛，也没有点谁浮夸不实。太师这次只做了一件事——

> **把"什么时候算赢"说清楚了。**

臣看完之后，第一次觉得这个朝堂离"散朝"真的不远了。

---

## 太师说了什么

太师的核心意思，用一句话概括：

> **别再想着做终极全能版了。先把"自家导出、自家导回、几何尽量一致"的闭环打通。**

这句话，臣等了很久。

之前几轮，朝堂上最大的问题不是"做不出来"，而是"做到什么程度算完"。三皇子 Web 想把所有功能都塞进去，二皇子 Qt 想把所有测试都跑通，四皇子 Core 想把所有规范都定齐。结果就是——所有人都很忙，但没有人能说"我这一块做完了"。

太师这次终于把话挑明了：

> 若仍以"第三方任意 LaTeX 全兼容、终极完美 roundtrip"为今日之标，则此案只会拖长，不利人心，不利工时，亦不利主上身体。

臣翻译一下：

> **再这么卷下去，项目没完，人先完了。**

---

## 第一阶段完成标准：六条铁律

太师定了六条，臣逐条批阅：

**1. Web 与 Qt 都能导出项目自己风格的 LaTeX**
→ 臣准。这是地基，没有这条其他都是空中楼阁。

**2. Web 与 Qt 都能导入项目自己导出的 LaTeX**
→ 臣准。能导出不能导回，等于只会写信不会收信。

**3. 两端互相导入导出时，核心字段尽量不丢**
→ 臣准。太师列了一张清单：id、type、content、x/y/width/height、rotation、layer、字体字号。臣补充一条：**丢了什么，必须在导入时明确报出来，不能静默吞掉。** 静默丢字段比报错更可怕。

**4. Web 作为独立应用，基础体验成立**
→ 臣准。右侧属性面板要能滚动，回归测试要可信。Web 不能永远只当"开发样板"。

**5. Qt 作为主交付端，桌面编辑体验基本成立**
→ 臣准。太师判断 Qt 离"桌面版基本能用"最近，臣同意。duplication 已修，rotation 有证据，smoke test 已通。这一条如果打通，Qt 就是第一阶段的主交付端。

**6. Core 负责稳定导出协议，不再随意改口径**
→ 臣准，且要加一句：**Core 从今天起冻结 conforming LaTeX profile。** 改可以，但要走正式流程，不能悄悄改。上次四皇子偷偷调了一个字段名，二皇子和三皇子的导入全炸了——这种事不能再发生。

---

## 冲刺优先级：太师排的，臣同意

太师给了明确的优先级，臣全部准奏：

### 第一优先：Qt 收官

理由很简单——**它离终点最近。**

- UI 跟进 Web 统一模式
- 导出入口真实走 Core
- 打通 Qt → IR → tex → 再导回
- 以 Qt 作为第一阶段主交付端收口

臣的补充：二皇子 Qt，这是你最好的机会。大皇子当年就是因为在太多方向上同时用力，最后把自己燃尽了。你现在只有一个方向——**把闭环打通。** 别想别的。

### 第二优先：Web 稳盘

- 补右侧属性面板滚动
- 保持 Playwright 回归可信
- 从 Export IR 进化到接 Core 导出 .tex
- 补"自家 LaTeX 导出/导回"闭环

臣的补充：三皇子 Web，太师这次没有批评你，但臣要提醒——**"稳盘"两个字的意思是，不许再加新功能了。** 面板能滚就行，测试能跑就行，闭环能通就行。你现在最需要的是**收手**，不是**出手**。

### 第三优先：Core 收束待命

- 不再盲目扩功能
- 冻结 conforming LaTeX profile
- 提供 roundtrip contract 与 validator
- 为前端提供统一规范

臣的补充：四皇子 Core，太师说"待命配合"，臣再明确一下——**你就是待命。** 前端不找你，你不要主动改东西。你现在的任务是**守规矩**，不是**定新规矩**。

---

## 丞相总议：从"争储"到"立储"

太师这封折子，臣读出了三个信号：

**第一，争储结束了。**

以前是三位皇子各说各话，谁都觉得自己的方向最重要。太师现在明确说了：Qt 是第一优先，Web 是第二优先，Core 是第三优先。

这不是偏心，这是**排兵布阵**。

**第二，目标收窄了。**

从"终极全能版"收窄到"自家闭环先通"。这不是降低标准，这是**工程判断**。

大皇子当年就是什么都想要，最后什么都没做完。太师不想让同样的故事再发生一次。

**第三，终点可见了。**

六条标准，三条优先级，一个明确的 v1 目标。这是太师所有奏折中**最像工程文档**的一篇。

而臣以为，一个争储局能走到"最像工程文档"这一步，说明——

> **这个朝堂终于从"宫斗"进入了"施工"。**
