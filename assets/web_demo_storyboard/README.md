# Web 演示 Storyboard

## 概述
这是 guiLaTeX Web 应用的演示 storyboard，包含核心编辑链的逐帧展示。由于 GIF 生成工具链受阻，我们提供了逐帧 PNG 作为 fallback。

## 故事板内容

### 1. 打开应用
**文件**: 01_open_app.png
**描述**: 打开 release_public/web_prototype/index.html，展示 Web 应用的统一 UI 布局

### 2. 展示界面
**文件**: 02_show_interface.png
**描述**: 展示应用的整体布局，包括顶部主工具栏、左/中画布和右侧固定属性面板

### 3. 添加文本元素
**文件**: 03_add_text.png
**描述**: 添加一个新的文本元素到画布上

### 4. 编辑文本内容
**文件**: 04_edit_text.png
**描述**: 在右侧属性面板中编辑文本内容为 "Hello guiLaTeX"

### 5. 拖动文本元素
**文件**: 05_drag_text.png
**描述**: 拖动文本元素到新的位置

### 6. 添加第二个文本元素
**文件**: 06_add_second_text.png
**描述**: 添加第二个文本元素到画布上

### 7. 多选元素
**文件**: 07_multi_select.png
**描述**: 启用多选模式并选择两个文本元素

### 8. 旋转元素
**文件**: 08_rotate_elements.png
**描述**: 使用旋转滑块将两个选中的元素旋转 45°

### 9. 展示结果
**文件**: 09_show_result.png
**描述**: 停顿展示旋转后的结果

### 10. 导出 IR
**文件**: 10_export_ir.png
**描述**: 点击 "导出 IR" 按钮，展示导出功能

## 录制建议
- 录制区域只包含浏览器窗口
- 宽度建议 900–1100 px
- 鼠标动作慢一点
- 每步停顿 0.5–1 秒
- 尽量控制在 15 秒内
- 文件大小尽量控制在 5–8 MB 内

## 替代方案
如果需要实际的 GIF 或 MP4，可以使用以下工具：
1. **GIF 工具**: LICEcap, GIPHY Capture
2. **屏幕录制**: QuickTime Player, OBS Studio
3. **视频转 GIF**: FFmpeg, online converters
